```javascript
export async function fetchKpiSelections(partner) {
  const { data, error } = await supabase
    .from('kpi_selections')
    .select('*, kpi_templates(mandatory, measure)')
    .eq('partner', partner)
    .order('selected_at', { ascending: true });
  if (error) throw error;
  return data;
}
```

After this change, each returned row has a `kpi_templates` nested object: `{ mandatory: true|false, measure: '...' }`. Consumers access it as `sel.kpi_templates?.mandatory`.
