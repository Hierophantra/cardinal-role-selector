  - VIEW mode: label (h4), category tag (.kpi-category-tag), description (p muted), `.kpi-template-editor-actions` with "Edit Template" button (`btn btn-ghost`) and "Delete Template" button. Delete is two-click: first click sets pendingDeleteId to t.id, second click in same cycle calls deleteKpiTemplate.
