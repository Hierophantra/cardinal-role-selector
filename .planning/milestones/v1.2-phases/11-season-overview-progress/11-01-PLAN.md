**Step 2: Create `src/lib/seasonStats.js`** with four named exports:

**`computeSeasonStats(kpiSelections, scorecards)`** — per D-04, INSGHT-01:
- Filter scorecards to those with `committed_at` (truthy).
- Initialize `hits = 0`, `possible = 0`, `perKpi = {}` keyed by `kpiSelections[i].id`.
- For each committed scorecard, for each kpi in kpiSelections: read `card.kpi_results?.[k.id]?.result`. If `'yes'`: increment both hits and possible (and per-kpi). If `'no'`: increment possible only. If `null` or missing: skip entirely (null-exclusion per STATE.md decision).
- `seasonHitRate = possible > 0 ? Math.round((hits / possible) * 100) : null`
- `perKpiStats` = array of `{ id, label: k.label_snapshot, hitRate: (s.possible > 0 ? Math.round((s.hits / s.possible) * 100) : null), hits: s.hits, possible: s.possible }` for each kpi.
- Return `{ seasonHitRate, perKpiStats }`.

**`computeStreaks(kpiSelections, scorecards)`** — per D-09, D-10, INSGHT-04:
- Filter scorecards to those with `committed_at`.
- Scorecards are already sorted newest-first by `fetchScorecards()`.
- For each kpi in kpiSelections: walk committed scorecards in order. Read `card.kpi_results?.[k.id]?.result`. If `'no'`, increment streak counter. On ANY other value (`'yes'` OR `null`), break immediately. This ensures null breaks a miss streak (Pitfall 3 from RESEARCH.md).
- Return array of `{ id: k.id, label: k.label_snapshot, streak: number }`.

**`computeWeekNumber()`** — per INSGHT-02:
- Import `SEASON_START_DATE` from `'../data/content.js'` and `getMondayOf` from `'./week.js'`.
- Parse SEASON_START_DATE: `const [sy, sm, sd] = SEASON_START_DATE.split('-').map(Number); const seasonStart = new Date(sy, sm - 1, sd);`
- Get current Monday: `const currentMonday = getMondayOf(); const [cy, cm, cd] = currentMonday.split('-').map(Number); const current = new Date(cy, cm - 1, cd);`
- Compute: `const diffMs = current - seasonStart; const diffWeeks = Math.floor(diffMs / (7 * 24 * 60 * 60 * 1000)); return Math.max(1, diffWeeks + 1);`
- CRITICAL: Use local-time Date construction (year, month-1, day) — NOT `new Date(string)` — matching the CRITICAL comment in week.js.

**`getPerformanceColor(hitRate)`** — per D-07:
- If `hitRate === null`, return `'var(--border)'`.
- If `hitRate >= 80`, return `'var(--success)'`.
- If `hitRate >= 50`, return `'var(--gold)'`.
- Otherwise return `'var(--miss)'`.

All functions are named exports. No default export. File header comment: `// Season stats computation helpers (Phase 11)`.
